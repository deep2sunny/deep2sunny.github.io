My Skill Set

HTML5 is the latest evolution of the standard that defines HTML. This technology will help me to write the content of a website in a very clear and concice manner. It is a new version of the language HTML with new elements which adds more semantic meaning to the code.

CSS3 is the latest evolution of the Cascading Style Sheets language and it aims at extending CSS2.1 by adding functionalities like transform and animation. This language will help me design websites by including layouts,colors, fonts and make it responsive as well.

Sass is a preprocessor scripting language that is interpreted or compiled into Cascading Style Sheets(CSS). This language will help me to write more clear and concise stylesheets by creating variables,mixins,media queries and functions. It helps in removing redundacy from code.

SQL stands for Structured Query Language. It is used to communicate with a database. I think it is must for a programmer to know a query language as it helps to create dynamic pages specially when it is combines with PHP and Node.js.I know how to work with MySQL and also design databases using EER Diagrams.

JavaScript is often abbreviated as JS, is a high level, interpreted programming language. It is a dynamic, weakly typed language. This technology will help me to add behaviour to a static website and also do data validation before sending information to the server.

Python is an interpreted high-level programming language. It provides constructs that enable clear programming on both small and large scales. Its simple syntax and open source structure helps me in writing high productivity and readable code in very less time.
